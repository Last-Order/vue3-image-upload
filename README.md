# vue3-image-upload

## 简介

这是一个使用 Vue3 + Vite 写的图片上传程序。

本项目为 Vue3 实践练手用项目，但其实还是有几个好玩的点。

- 支持选择文件上传、拖拽上传、剪贴板上传
- 支持自动复制链接（Clipboard API）
- Web Share API
- PWA（随便设置了一下
